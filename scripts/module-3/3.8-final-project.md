## 🔨 SECTION 3: AI-Enhanced Workflows (40 นาที)

### สร้าง 3 Workflows ที่ใช้งานจริง

**Workflow #1: Research Workflow**
```markdown
## 🔍 UX Research with AI

**Trigger:** เมื่อต้องทำ User Research

**Steps:**
1. Define research questions
   └─ AI: ช่วย refine และเพิ่ม edge cases

2. Create interview guide
   └─ AI: Generate questions + follow-ups
   └─ Prompt: "Create user interview guide for [topic]..."

3. Analyze interview notes
   └─ AI: Extract themes และ patterns
   └─ Prompt: "Analyze these notes and identify..."

4. Create insights report
   └─ AI: Structure findings + recommendations

**Time Comparison:**
- Without AI: 16 hours
- With AI: 6 hours
- Savings: 10 hours (62%)
```

**Workflow #2: Content/Writing Workflow**
```markdown
## ✍️ UX Writing Workflow

**Trigger:** เมื่อต้องเขียน UI copy

**Steps:**
1. Gather context
   └─ AI: Summarize requirements

2. Generate variations
   └─ AI: 5 variations per element
   └─ Prompt: "Write 5 variations of [element]..."

3. Review & select
   └─ Human: Choose best options

4. Localization
   └─ AI: Translate + adapt tone
   └─ Human: Final review

**Quality Check:**
- [ ] Brand voice consistent?
- [ ] Clear และ concise?
- [ ] Accessible language?
```

**Workflow #3: Design Iteration Workflow**
```markdown
## 🎨 Design Iteration Workflow

**Trigger:** เมื่อต้องทำ design variations

**Steps:**
1. Design first version
   └─ Human: Core design decisions

2. Generate alternatives
   └─ AI: Color, layout, copy variations
   └─ Tools: Figma AI, Midjourney

3. Get AI feedback
   └─ AI: Critique based on heuristics
   └─ Prompt: "Evaluate this design for..."

4. Iterate based on feedback
   └─ Human: Final design decisions

**Time Comparison:**
- Without AI: 8 hours
- With AI: 3 hours
- Savings: 5 hours (62%)
```

## 🔨 SECTION 4: Personal Prompt Library (30 นาที)

### รวบรวม 10+ Prompts ที่ใช้บ่อย

**Template:**
```markdown
## 📝 My Prompt Library

### 🔍 Research Prompts

**R1: User Interview Questions**
```
I'm researching [topic] for [product type].
Target users: [description]
Research goals: [goals]

Create 10 interview questions that:
- Start broad, then get specific
- Include follow-up prompts
- Avoid leading questions
```

**R2: Competitive Analysis**
```
Analyze [competitor] as a UX professional:

1. First impressions (homepage)
2. Key user flows (3 main journeys)
3. Design patterns used
4. UX strengths (3 points)
5. UX weaknesses (3 points)
6. What we can learn
```

### ✍️ Writing Prompts

**W1: Microcopy Generator**
```
Context: [app type], [screen name]
User state: [what user just did/will do]
Brand voice: [tone description]

Write 5 variations for [element]:
- Max [X] characters
- Action-oriented
- Friendly but professional
```

**W2: Error Message**
```
Situation: [what went wrong]
User impact: [how it affects them]
Brand tone: [description]

Create error message with:
- What happened (1 sentence)
- Why it matters (optional)
- What to do next (clear action)

## 🔨 SECTION 5: Ethics Framework (15 นาที)

### สร้าง Personal Guidelines

```markdown
## ⚖️ My AI Ethics Framework

### ✅ I WILL:
- [ ] Disclose AI use to stakeholders when appropriate
- [ ] Always review and verify AI outputs
- [ ] Credit AI tools in documentation
- [ ] Use AI to augment, not replace, critical thinking
- [ ] Stay updated on AI ethics developments

### ❌ I WON'T:
- [ ] Use AI-generated content without review
- [ ] Input confidential client data without permission
- [ ] Claim AI work as purely my own creation
- [ ] Use AI for final decisions on sensitive topics
- [ ] Ignore AI biases and limitations

### 🟡 GRAY AREAS (Case-by-case):
- Using AI for competitive research
- AI-generated images in client work
- Sharing prompts with other tools

### 📋 DISCLOSURE POLICY:
"In my work, I use AI tools to [accelerate research / 
generate initial drafts / brainstorm ideas]. All AI-assisted 
work is reviewed and refined by me to ensure quality 
and accuracy."

### 🔒 DATA HANDLING:
- Client names: Never input to AI
- User data: Anonymize first
- Proprietary info: Check NDA first
- Credentials: Never share with AI
```
